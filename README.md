# Community WebView Frontend

## Run locally

```bash
npm install
npm run dev
```

App starts on `http://localhost:3001`.

## Quality checks

```bash
npm run lint
npm test
npm run test:ci
```

## Notes

- Jest setup file: `src/__tests__/setupTests.ts`
- API helpers live in: `lib/api/`
- App Router pages live in: `app/`
